## How to Update Videos

### Replace a Video

1. Click on the `videos` folder above
2. Click on the video file you want to replace
3. Click the trash icon (üóëÔ∏è) to delete it
4. Go back to the `videos` folder
5. Click "Add file" ‚Üí "Upload files"
6. Upload your new video with the **exact same filename**
7. Click "Commit changes"

**Important**: Keep the same filename so the website continues to work!

### Add a New Testimonial Video

1. Name it `testimonial-11.mp4` (or the next number)
2. Upload it to the `videos` folder
3. Contact your developer to add it to the website

Your developer will need to update the Webflow code to display new videos.

## Video Requirements

- **Format**: `.mp4` (H.264 codec)
- **Aspect ratio**: 9:16 (vertical)
- **File size**: Under 50MB recommended (under 100MB max)
- **Resolution**: 1080x1920 or 720x1280
- Shorter/compressed videos load faster for users

## Important: Keep This Repository Public

This repository must remain **Public** for videos to work on your website.

If you make it Private, videos will stop loading.
